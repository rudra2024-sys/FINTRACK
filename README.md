# 🚀 FinTrack — Finance Management System

## 📌 Overview

FinTrack is a full-stack finance management system with secure APIs, budgeting, transactions, and analytics.

## 🏗️ Tech Stack

* Backend: Spring Boot (JWT, REST APIs)
* Frontend: HTML / JS UI
* Database: PostgreSQL
* DevOps: Docker

```id="e8mp5o"

frontend/       → UI

```
