# 🚀 FinTrack — Finance Management System

## 📌 Overview

FinTrack is a production-ready finance management system built with Spring Boot, PostgreSQL, and Docker. It allows users to track expenses, manage budgets, and analyze financial data.

## 🏗️ Tech Stack

* Backend: Spring Boot (JWT Authentication)
* Frontend: HTML / JS
* Database: PostgreSQL
* DevOps: Docker & Docker Compose

## 📁 Project Structure

```
backend/        → Spring Boot backend
frontend/       → UI
docker-compose.yml
```

## ⚙️ How to Run

```bash
docker-compose up --build
```
